# 🎯 Telegram Funstat Parser Bot + Пробив

**Многофункциональный Telegram-бот** для парсинга чатов и пробива пользователей по ID или username.

---

## ✨ Возможности

- 🔍 **Пробив** — поиск пользователя + вся история его сообщений по чатам
- 📥 **Парсинг** — сбор участников и сообщений из Telegram-чатов
- 💎 Система кристаллов + реферальная система
- 🎮 Мини-игра «Угадай число»
- 📊 Личная статистика и топ пользователей
- 🛠 Админ-панель (парсинг, импорт, баны, выдача кристаллов)

---

## 🚀 Быстрый старт

### Шаг 1: Скачай проект
```bash
git clone https://github.com/ТВОЙ_НИК/НАЗВАНИЕ_РЕПО.git
cd НАЗВАНИЕ_РЕПО
```

### Шаг 2: Установи зависимости
```bash
pip install -r requirements.txt
```
### Шаг 3: Настрой токены
```bash
Bashcp .env.example .env
cp config.example.json config.json
```
#### Открой файл .env и вставь свои данные (BOT_TOKEN, API_ID, API_HASH и т.д.).

### Шаг 4: Авторизация в Telegram
```bash
python authorize.py
```

### если есть вопросы пишите LLM и если есть кто то кто поможет то пишите ему/ей;
### к сожалению на время не имею контактов 
Введи код подтверждения, который придёт на телефон.
```


### Шаг 5: Запусти бота
```bash
Bashpython main.py
```
